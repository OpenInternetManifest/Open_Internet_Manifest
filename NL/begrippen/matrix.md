# Matrix

Open, gedecentraliseerd communicatieprotocol voor chat, voice en video.  
Iedereen kan een eigen homeserver (node) draaien — precies zoals e-mail.

Synapse, Dendrite of Conduit op een Raspberry Pi is genoeg voor een gezin of hele buurt.  
Geen WhatsApp, geen Telegram, geen datagraaiende tussenpartij.
