# IPFS (InterPlanetary File System)

Decentraal bestandssysteem dat werkt op basis van content-adressering (CID) in plaats van locatie.  
Een bestand dat eenmaal op IPFS staat, blijft voor altijd beschikbaar zolang één node het bewaart.

Geen enkele server kan het verwijderen.  
Geen CDN nodig.  
Geen cloudrekening.
