# Umbrel

Gebruiksvriendelijk operating system voor Raspberry Pi of dedicated hardware – maakt self-hosting van Bitcoin node, Nextcloud, Matrix, etc. eenvoudig.

Alles via een mooie web-interface, plug-and-play.
