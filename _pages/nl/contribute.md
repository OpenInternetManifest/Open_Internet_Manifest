  <div class="how-it-works">
    <h2>Hoe werkt bijdragen?</h2>
    <ol>
      <li>Kies hierboven het type bijdrage en taal.</li>
      <li>Vul het eenvoudige formulier in (titel, teaser en je markdown tekst).</li>
      <li>Een GitHub Action maakt automatisch een Pull Request met het juiste nummer en placeholders.</li>
      <li>Het core team reviewt je bijdrage (meestal binnen 48 uur).</li>
      <li>Bij goedkeuring wordt alles automatisch ingevuld (hashes, git info, etc.) en gepubliceerd.</li>
    </ol>
