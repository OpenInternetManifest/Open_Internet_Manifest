  <div class="how-it-works">
    <h2>How does contributing work?</h2>
    <ol>
      <li>Choose the type of contribution and language above.</li>
      <li>Fill in the simple form (title, teaser and your markdown text).</li>
      <li>A GitHub Action creates a Pull Request with the correct number and placeholders.</li>
      <li>The core team reviews your contribution (usually within 48 hours).</li>
      <li>After approval, everything is automatically filled in (hashes, git info, etc.) and published.</li>
    </ol>
