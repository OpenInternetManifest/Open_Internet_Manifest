### RVN: Deel 4 – Vervang Windows (je computer is niet van jou) 💻

**Narratief:**  
“Windows is gewoon het makkelijkst. Op Linux kan je toch niet alles draaien.”

**Realiteit:**

Ja, dat was 10-15 jaar geleden grotendeels waar.  
Vandaag de dag is dat verhaal grotendeels **achterhaald**.

**Hoe zit het echt in 2026?**

- **99% van de dagelijkse dingen** werkt uitstekend op Linux (browser, mail, Office-documenten, video, foto’s, bankieren, etc.)
- **Gaming**: Steam + Proton werkt inmiddels extreem goed. Veel games draaien zelfs beter dan op Windows.
- **Professioneel werk**: Photoshop → Alternatieven zoals Affinity, GIMP + Photopea.  
  Adobe Suite → Kan via virtual machine of webversies.  
  AutoCAD / specialistische software → Vaak nog een struikelblok (maar vaak ook virtual machine oplossing).
- **Bankieren & overheid**: Werkt in de meeste gevallen prima via browser.

**De eerlijkheid gebiedt te zeggen:**
Er zijn nog een paar niche programma’s die alleen goed op Windows draaien. Voor die gevallen kun je simpel een **dual-boot** houden of Windows in een virtuele machine draaien (via VirtualBox of virt-manager).

**Makkelijkste opties voor beginners:**

- **Linux Mint** → Beste keuze als je van Windows komt (ziet er bekend uit)
- **Zorin OS** → Lijkt nog meer op Windows
- **Pop!_OS** → Uitstekend voor gaming en moderne hardware

**Actie voor vandaag / dit weekend:**

1. Download Linux Mint → https://linuxmint.com
2. Maak een bootable USB-stick
3. Probeer het eerst in **Live-modus** (zonder Windows te verwijderen)
4. Kijk of jouw belangrijke programma’s werken

Je hoeft Windows niet meteen weg te gooien. Dual-boot is prima.

**OIM-uitweg**

Digitale soevereiniteit betekent niet dat alles perfect moet werken vanaf dag 1.  
Het betekent dat je bewust kiest voor systemen waar **jij** de baas bent, in plaats van Microsoft.

Dit was het laatste deel van **Fase 1** (de basisstappen).

**Vraag aan jou**

Wat houdt jou nog tegen om Linux te proberen?  
Is het “ik kan niet alles draaien” of iets anders?

Deel gerust je bezwaren of ervaringen.
